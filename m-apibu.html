Introducing Monitor+

Monitor+ is the Web API that provides easy access to a huge range of historical reference market data from stock exchanges, commodity exchanges, proprietary market data providers, independent pricing agencies as well as currencies, global indices and more.

The Monitor+ API sits between the subscriber and the cloud market data servers and enables subscribers to request and receive time series market data in JSON format.

With simple flexible licensing, Monitor+ allows easy integration of market data into new or existing platforms, information systems and trading systems. The Monitor+ API is easy to use, extremely powerful and has native access in both Python and "R" which enables data scientists, analysts and developers to use it straight out of the box.

How it Works

A clients subscribes to one or more datasources and receives an API key to access the service. The client sends a HTTP request for market data and the server returns the data in JSON format. It's really that simple.

With just two HTTP requests you can authenticate and get values for any number of datasets (time series) that your subscription permits.

The API is well documented with interactive help in both "R" Studio and Python. We provide a lot of sample code in the documentation as well as working examples in both "R" and Python.

The API also provides easy access to the metadata (data that describes the datasources and datasets) and the actual values as well managing as list of "Favorite" datasets for each user.

Additional Features

JSON Data Formats

Monitor+ is ideal for creating a variety of analytic applications and reports as market data can be returned in one of three JSON formats:

  • Single Date. Ideal for creating "quote page" style reports that require values for one date with optional "Spark Line" and "Change" values
  • Historical TS. Market data is returned as individual time series.
  • Historical RC. Market data is returned in a ”row x column” format to make report creation easy.

Integration

Our market data is natively accessible in both "R" and Python. You don't have to use "R" or Python as almost any other development language such as C#, Java or PHP can be used to call the API directly.

Monitor+ should be easy to integrate into new asnd existing systems as the API is designed to be return large amounts of metadata and time series data as well as managing a list of user "Favorite" datasets with makes user times series selection simple and easy. Excel users can use our XLerate for Excel add-in to create, update and share detailed Excel market reports. Details about XLerate for Excel can be found on the "Products” menu at the top of the page.

Averages

The API can return values as daily, weekly, half monthly, monthly, quarterly, half yearly and yearly. Custom averages can be calculated for weeks, half months and months. Custom quarterly, half-yearly and yearly averages can be created using custom month averages.

Data Preparation

Monitor+ can perform data preparation for you. Choose the date range, average, exclude (or move) weekend values and choose to use the first or last date in averages. Fill missing values using previous, mid point or interpolated values and choose when to apply any fill. If data vendor supported, we apply and return details of any published corrected values.
