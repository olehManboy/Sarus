Market data made easy

Overview:

Monitor+ is a Web API that allows easy access to a huge range of market data from cloud data servers that store historical reference data from stock exchanges, commodity exchanges, proprietary market data providers, independent pricing agencies as well as currencies, world indices and more.

The Monitor+ API site between the subscriber and the cloud data servers enabling users to request and receive time series market data in standard JSON format.

With simple flexible licensing, Monitor+ allows easy integration of market data into new or existing platforms, information systems and trading systems. It is easy to use, extremely powerful and has native access in both Python and "R" to enable data scientists, analysts and developers to use it straight out of the box.





How it works:

A clients subscribes to one or more datasources and receives an API key that gives access to the service. The client sends a HTTP request for data and the server returns the market data in JSON format. It's that simple. With just two HTTP requests you can authenticate with the server and get values for any number of datasets that your subscription permits.

The API is well documented with sample code and provides both easy access to the metadata (data that describes the datasources and datasets) and the actual values.

Additional API Features:

Result Data Format: Monitor+ is ideal for creating a variety of analytic applications and reports as market data can be returned in one of three JSON formats:

  • Single Date. Ideal for creating "quote page" style reports that require values for one date with optional "Spark Line" and "Change" values.

  • Historical TS. Market data is returned as individual time series.

  • Historical RC. Market data is returned in a ”row x column” format to make report creation easy.

Integration: The market data is directly accessible in "R" and Python or using our XLerate for Excel add-in. Use XLerate to create, update and share detailed Excel market reports. See the “Products” menu above for detailed information on XLerate for Excel. You can get more information on the “R” , Python packages and the API from the “Documentation” menu at the top of the page.

Averages: The API can return values as daily, weekly, half monthly, monthly, quarterly, half yearly and yearly. Custom averages can be calculated for weeks, half months and months. Custom quarterly, half-yearly and yearly averages can be created using custom month averages.

Data Preparation: Choose the date range and opt to: exclude (or move) weekend values, adjust weekend dates in average results and use the first or last date for the average. Fill options permit the filling of missing values with the previous, mid point or interpolated values. If the data vendor supports the feature, we apply and return details of any published corrected values in the result.
