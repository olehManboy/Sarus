The Market Data Analysis Software

Overview

Sarus XLerate is a powerful, easy to use Excel add-in that can provide a comprehensive end-of-day market data solution for your trading and analytic needs. It integrates seamlessly into Excel enabling you to create amazing market data reports with just a few mouse clicks - and from virtually anywhere, 24/7.

There are no cryptic exchange symbols to remember and you can create detailed market data charts that allow zoom and scroll with a single mouse click. Controlling the chart is simple with the XLerate built in dialogs.

There are over 40 built-in analytic “Quick functions” that can be added to any data column for further detailed analysis. Create reports in any frequency - day to year (including custom weeks, half months and months). Choose date ranges, fill type, lead/lag, how weekends are handled and much more.

Sarus XLerate gives unprecedented functionality and control and is suited to front, mid or back office roles where it can be used with ease by traders, analysts, data scientists and office staff.

If you can use Excel, you can use XLerate.

Reports

XLerate has three main report types:

  • Time Series - select, display and chart any datasets in standard Excel column format.
  • Analytic - predefined interactive reports to help you quickly analyse and compare datasets.
  • Snapshot - build quote style pages showing a snapshot of any market on a particular date.

All reports are saved as Excel workbooks to make sharing and updating by colleagues easy. Each report type is explained in more detail in the sections below.

Time Series Charts

XLerate has beautifully detailed charts that can be resized, zoomed and scrolled along the date axis. You can plot any worksheet data series along the left or right value axis allowing worksheet series with no obvious price relationship to be overlaid along same date axis.

Charts and plots can be edited with the built in XLerate dialogs and can be split allowing four price axis per chart. The chart date axis was designed for market data analysis and aligns on week, month and year boundaries.

Analytic Reports

XLerate has a number of built in interactive analytic reports and studies to help you quickly analyse datasets. They include Time Series Analysis, Correlation Matrix, FX, Seasonality, ANOVA and MACD. Each report contains three worksheets source, chart and report.

Analytic report charts can also be edited, scrolled and zoomed. You can choose which columns to display in each. Many charts can display a data panel showing the chart values (see the top of the image below).

Snapshot Reports

Snapshot reports allow you to build your own quote pages containing a number of contracts from various datasources. The reports can contain average results as well as "Change" and "Sparkline" data. The contracts for the report are stored in a source worksheet together with the API result data and any sparkline values. You then create the display worksheet and reference the source sheet value cells. If the sample report below, we show sample sparklines and have added the option to allow the user to select contracts (using checkboxes) and to create a historical report using the selected contracts. We provide sample code and a clear explanation on how to create your own.



As well as fees for proprietary data, many data vendors have additional fees for their published reports. If you subscribe to the vendors reports and their data service, it may be possible to recreate these published reports using snapshot reports. You can replicate and improve on the vendors report price tables.

Snapshot reports are always up to date with any published corrections (at the time the reports were created or refreshed). They can be created for virtually any date with the click of a button. We can help you get started with sample VBA code and advice.


Quick functions

XLerate has over 40 drop in “Quick Functions” ("QF") that allow you easily add analytic columns to your worksheet data columns. These analytic functions are activated directly from the Excel pop-up menu. With the QF editor, just select the source columns, click Ok and the QF is inserted in a column next to the source column. Best of all the QF's can be edited and they update when the worksheet refreshes.
